### Linux

Works also with NPM!

sudo apt install npm

// choose version 14 or higher, tested only for version 14
curl -fsSL https://deb.nodesource.com/setup_14.x | sudo -E bash -

// go to folder gcal

npm install electron-builder
npm run dist
